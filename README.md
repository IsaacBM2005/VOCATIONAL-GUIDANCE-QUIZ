# VOCATIONAL-GUIDANCE-QUIZ
Herramienta interactiva de orientación vocacional para estudiantes de preparatoria. Incluye 40 preguntas estructuradas en 8 dimensiones (intereses, habilidades, valores, contexto, proyección futura y análisis FODA), con diagnóstico personalizado generado por Inteligencia Artificial y recomendaciones de carrera adaptadas al perfil del estudiante.
